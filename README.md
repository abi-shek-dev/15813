# Afford-Med Campus Evaluation Notification App

## Tech Stack I Used
- **React 19**: For building the user interface.
- **Vite**: Because it's way faster for development.
- **Vanila CSS**: Because I dont know Material UI rather i know tailwindcss

## How to run this locally

If you want to test it out on your machine, just follow these steps:

1. First, make sure you install all the required packages by running: `npm install`
2. Then, you can start the local development server with: `npm run dev`
3. If you need to build it for production, use: `npm run build`
